# MaaInspector 更新日志

## 1.2.0 (2025-01-19)

### 新功能
- 🎨 新增应用设置面板,支持个性化配置
- 📢 新增更新公告功能,及时了解最新动态
- 🖼️ 优化设备截图预览性能
- ⚙️ 支持自定义画布默认配置(连线类型、节点间距)

### 优化
- ⚡ 提升画布加载速度
- 🎯 改进节点搜索体验
- 📱 响应式布局优化
- 🔧 优化InfoPanel收起/展开动画流畅度

### 修复
- 修复资源文件切换时的状态保存问题
- 修复Win32设备连接参数配置bug
- 修复节点删除时的状态同步问题

## 1.1.0 (2025-01-15)

### 新功能
- 🔧 支持Win32窗口设备连接
- 🎛️ 新增多种截图和输入方法配置
  - 截图方法: GDI, FramePool, DXGI_DesktopDup等
  - 鼠标方法: Seize, SendMessage, PostMessage等
  - 键盘方法: Seize, SendMessage, PostMessage等
- 📸 新增设备实时截图预览功能
- 🔍 设备搜索支持ADB和Win32两种类型

### 优化
- ⚡ 优化节点属性编辑器性能
- 📝 改进流程图导出功能
- 🎨 统一UI组件样式
- 🌐 改进错误提示信息

### 修复
- 修复ADB设备连接超时问题
- 修复节点删除时的状态同步bug
- 修复资源路径配置保存失败问题

## 1.0.0 (2025-01-10)

### 新功能
- ✨ 首次正式发布
- 🎨 可视化拖拽节点编辑器
- 📊 内置流程图自动布局算法
- 🔍 实时节点调试功能
- 📂 资源文件管理(加载、切换、保存)
- 🔌 支持ADB设备连接
- 🤖 Agent执行控制
- 🖼️ 模板图像管理

### 特性
- 基于Vue 3 + Tauri构建
- 支持MaaFramework 5.0+ Pipeline
- 轻量级跨平台桌面应用
- 实时查看节点执行状态
- 节点属性双击编辑
- 右键菜单快捷操作
